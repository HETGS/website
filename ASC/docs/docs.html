Memos and specifications

We provide a range of information about pipeline processing, the CXC Data Analysis System, Grating related analysis issues, and general scientific analysis algorithms.

The MIT/CXC team produces different types of documents:

  • spec (specification) Exact specification for the behavior of analysis tools. Look here for a description of the exact algorithm that a CIAO tool should implement.
  • ARD (analysis reference data) Specification for the mormat of CALDB files
  • ICD (Interface control document) Specification of file format produced by tool.
  • memo General documents written to support the development of specs, ARDs, or ICDs. For example, the analysis of a specific detector effect might be documented here. If that analysis leads to specific changes in the algorithm for a tool, then the new algorithm will be included into the revised version of the relevant specs.
  • misc Miscellaneous files including white papers and presentations.

The documents listed on this page are revised over time, some may still be under development, while others may become obsolute when a tool is discontinued, e.g. because its functionality is now integrated into other tools.
